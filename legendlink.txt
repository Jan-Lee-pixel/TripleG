## 📝 Admin Approval Workflow

### For New Admin Users:
1. Register → http://127.0.0.1:8000/accounts/admin-auth/register/
2. Verify Email → http://127.0.0.1:8000/accounts/admin-auth/verify-otp/
3. Get Approved → Use Django Admin at http://127.0.0.1:8000/admin/
4. Login → http://127.0.0.1:8000/accounts/admin-auth/login/
5. Access Dashboard → Redirects to http://127.0.0.1:8000/portfolio/projectmanagement/

### For Existing Admin Users:
1. Login → http://127.0.0.1:8000/accounts/admin-auth/login/
2. Access Dashboard → Redirects to http://127.0.0.1:8000/portfolio/projectmanagement/

## 🔑 Key Notes:
- Admin accounts require approval before login
- All URLs use OTP email verification
- Custom admin login avoids Django's built-in admin conflicts
- Post-login redirects to portfolio project management
- Superuser access via http://127.0.0.1:8000/admin/
