PROJECT VIDEO INSTRUCTIONS FOR TRIPLE G BUILDHUB

Place your project background video file in this directory with the name "projects-background.mp4" to use as the background for the projects section.

Requirements:
1. Format: MP4 (H.264 codec recommended)
2. Resolution: At least 1920x1080 (HD)
3. Duration: 15-60 seconds (will loop automatically)
4. Content: Construction footage, architectural renderings, or time-lapse footage that represents Triple G BuildHub projects
5. File size: Keep under 15MB for optimal loading performance

The video will autoplay, be muted, and loop continuously. It will have a dark overlay to ensure text readability.

If you need to change the video filename or add alternative formats (WebM, etc.), update the source tags in the projects.html file.
