VIDEO INSTRUCTIONS FOR TRIPLE G BUILDHUB

Place your video file in this directory with the name "construction-timelapse.mp4" to use as the background for the blog header.

Requirements:
1. Format: MP4 (H.264 codec recommended)
2. Resolution: At least 1920x1080 (HD)
3. Duration: 10-30 seconds (will loop automatically)
4. Content: Construction timelapse, aerial views of projects, or similar content that represents Triple G BuildHub
5. File size: Keep under 10MB for optimal loading performance

The video will autoplay, be muted, and loop continuously. It will have a dark overlay to ensure text readability.

If you need to change the video filename or add alternative formats (WebM, etc.), update the source tags in the bloglist.html file.
