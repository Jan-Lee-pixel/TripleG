PROJECT DETAIL VIDEO INSTRUCTIONS FOR TRIPLE G BUILDHUB

Place your project detail background video file in this directory with the name "project-detail.mp4" to use as the background for the project detail hero section.

Requirements:
1. Format: MP4 (H.264 codec recommended)
2. Resolution: At least 1920x1080 (HD)
3. Duration: 10-30 seconds (will loop automatically)
4. Content: Footage of the specific project showcased on the page
           (construction process, walkthrough, or aerial views)
5. File size: Keep under 10MB for optimal loading performance

The video will autoplay, be muted, and loop continuously. It will have a dark overlay to ensure text readability.

If you need to change the video filename or add alternative formats (WebM, etc.), update the source tags in the project-detail.html file.
